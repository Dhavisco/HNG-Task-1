# HNG Internship - Stage 1 Frontend Task

This README provides an overview of the Stage 1 frontend task completed as part of the HNG Internship.

## Task Description

The Stage 1 task involved creating a single-page website using HTML, CSS, and JavaScript. The website includes the following elements:

1. Displaying my Slack name prominently.
2. Showcasing my Slack profile picture.
3. Indicating the current day of the week.
4. Displaying the current UTC time in milliseconds.
5. Mentioning my track (e.g., Frontend).
6. Providing a clickable link to my GitHub repository.


## Project Structure

The project structure consists of the following files and folders:

- `index.html`: The main HTML file containing the webpage structure.
- `style.css`: The CSS file for styling the webpage.
- `script.js`: The JavaScript file to add interactivity and dynamic content.

## Result
The website is hosted on Github pages
https://dhavisco.github.io/HNG-Task-1/

## How to Use

1. Clone or download the repository to your local machine.

2. Open `index.html` in your web browser to view the single-page website.
